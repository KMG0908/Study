### 기본 타입(Primitive value)
#### 1. Boolean
**true**와 **false**의 두 가지 값을 가질 수 있음
#### 2. Null
**null**값만 가질 수 있음
#### 3. Undefined
값을 할당하지 않은 변수
#### 4. Number
#### 5. String
#### 6. Symbol
**유일**하고 **변경 불가능**한 기본값
